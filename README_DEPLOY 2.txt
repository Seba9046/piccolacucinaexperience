PICCOLA CUCINA EXPERIENCE - PACCHETTO SITO

Caricare tutto il contenuto di questa cartella nella root del server di piccolacucinaexperience.com oppure nella cartella /menu.

File principali:
- index.html: sito menu
- data.js: database piatti, ingredienti, allergeni, traduzioni
- styles.css: grafica stile siciliano
- script.js: filtri, lingue, ricerca, call waiter
- assets/video/taormina-intro.mp4: video intro generato
- assets/qr/piccola-cucina-experience-menu-qr.png: QR code verso https://piccolacucinaexperience.com/menu

Per aggiungere foto:
1. Inserire le immagini in assets/photos/
2. Aggiungere il riferimento nel singolo item dentro data.js.

Nota allergeni: il sito aiuta la consultazione, ma il cliente deve sempre informare lo staff prima di ordinare.
